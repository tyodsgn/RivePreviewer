# Rive Collection

A web app to store and preview your Rive animations. Upload `.riv` files via Admin, view them in a gallery with hover-to-animate and click-for-preview.

## Features

- **Gallery** – Grid of all Rive files, hover to animate, click for larger preview with controls
- **Admin** – Upload and delete Rive files (login: `admin` / `admin`)
- **Storage** – All files stored locally in IndexedDB (browser storage)

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
